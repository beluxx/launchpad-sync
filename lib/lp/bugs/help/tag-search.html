Bug tag search

You can get a list of bugs depending on the tags that those bugs have or don't have.

Next to the Tags search box are two radio buttons: Any and All. Select Any and Launchpad will return bugs that match any part of your search term. Select All for bugs that match your entire search query.

To search for the absence of a particular bug tag, place a minus sign before its name: e.g. -test

Examples

  • Find bugs with the test tag: search for test
  • Find bugs that don't have the test tag: search for -test
  • Find bugs that have both the test and ui tags: search for test ui and select the All radio button.
  • Find bugs that have either the test tag or do not have the ui tag: search for test -ui and select the Any radio button.
  • Find all bugs with one or more tags: search for *
  • Find all bugs that have no tags: search for -*

You can mix the different types of search.
